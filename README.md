# CRUD Full Stack (Node.js + React)

Este é um projeto full stack de CRUD (Create, Read, Update, Delete) desenvolvido com Node.js no backend e React no frontend.

O objetivo do projeto é demonstrar a integração entre uma API REST e uma interface web, permitindo o gerenciamento completo de dados.

## Tecnologias utilizadas

- Node.js
- Express
- React
- Axios
- Banco de dados (MySQL)

## Funcionalidades

- Criar novos registros
- Listar registros existentes
- Atualizar informações
- Excluir registros

## Estrutura do projeto

```
crud/
├── api/        # Backend (API REST com Node.js)
├── frontend/   # Interface web com React
```

## Objetivo

Este projeto foi desenvolvido para fins de estudo e prática de desenvolvimento full stack, abordando conceitos de API, consumo de dados e integração entre frontend e backend.
